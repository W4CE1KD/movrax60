# **App Name**: TestPilot

## Core Features:

- Question Input: Allow users to input test questions and answers via a simple form.
- Answer Evaluation: Use AI to automatically evaluate short-answer questions for correctness and completeness, acting as a grading tool.
- Score Calculation: Tally the score after the test.
- Display results: Show users the correct and incorrect answers at the end.

## Style Guidelines:

- Primary color: Dark Indigo (#3F51B5) to evoke trust and focus.
- Background color: Very light Lavender (#EDE7F6) provides a gentle backdrop.
- Accent color: Teal (#009688) is used for interactive elements.
- Body and headline font: 'Inter', a sans-serif font that ensures readability.
- Use simple, clear icons from a set like Material Design Icons for tests.
- A clean, single-column layout that keeps the focus on the content. Ample whitespace to avoid a cluttered look.
- Use subtle animations for transitions between questions, feedback and submission, etc.
