# **App Name**: ShadowHunters CTF

## Core Features:

- Animated Hero Section: Full-screen hero section with floating particles, energy waves, parallax movement, and cinematic logo animation.
- CTF Achievements Stats: Display animated counters for CTFs Attended, Archived, and Won, with glowing card hover effects.
- Upcoming CTFs Banner: Show upcoming CTFs in horizontal banner cards with smooth entrance animations and glowing hover effects.
- CTFTime Rank Display: Dramatic display of current CTFTime rank with animated reveal and background aura ring.
- Team Members Grid: Responsive grid layout of team member cards with 3D tilt, glow border animation, and navigation to member profile.

## Style Guidelines:

- Background color: Deep black (#000000) to set a dark, immersive tone.
- Primary color: Shadow purple (#645CAA) for a mysterious and powerful aesthetic.
- Accent color: Electric blue (#7DF9FF) to create glowing highlights and neon effects.
- Font: Use '/public/fonts/' globally for a consistent and custom look. Note: currently only Google Fonts are supported.
- Scroll-triggered animations, staggered reveals, magnetic hover effects, parallax layers, smooth spring motion, glow intensity interpolation, particle movement background, and energy pulse effects.
- Single-page layout without a navbar to emphasize a focused and cinematic experience.
- Use glowing shadows, particles, and aura effects to enhance the dark energy/shadow monarch aura.
